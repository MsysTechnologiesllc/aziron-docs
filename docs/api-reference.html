API Reference

Welcome to the Aziron API Reference documentation.

Overview‚Äã

The Aziron API provides comprehensive access to all multi-agent functionality, including:

  • Agent creation and management
  • Real-time chat and streaming
  • Workflow orchestration
  • Vector database operations
  • User authentication and authorization

Getting Started‚Äã

Base URL‚Äã

http://localhost:8080

Authentication‚Äã

All API requests require authentication using Bearer tokens:

curl -H "Authorization: Bearer YOUR_TOKEN" \
  http://localhost:8080/api/v1/agents

Interactive API Explorer‚Äã

For a complete interactive API reference with live testing capabilities, we'll be adding Swagger UI integration soon.

Core Endpoints‚Äã

Authentication‚Äã

  • POST /api/v1/auth/login - User authentication
  • POST /api/v1/auth/register - User registration
  • POST /api/v1/auth/refresh - Token refresh

Agents‚Äã

  • GET /api/v1/agents - List all agents
  • POST /api/v1/agents - Create new agent
  • GET /api/v1/agents/{id} - Get agent details
  • PUT /api/v1/agents/{id} - Update agent
  • DELETE /api/v1/agents/{id} - Delete agent

Chat & Streaming‚Äã

  • POST /api/v1/chat - Send chat message
  • GET /api/v1/chat/stream - WebSocket streaming endpoint
  • GET /api/v1/chat/{sessionId}/history - Get chat history

Workflows‚Äã

  • GET /api/v1/workflows - List workflows
  • POST /api/v1/workflows - Create workflow
  • POST /api/v1/workflows/{id}/execute - Execute workflow

Examples‚Äã

Create an Agent‚Äã

curl -X POST http://localhost:8080/api/v1/agents \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Assistant",
    "description": "A helpful AI assistant",
    "model": "gpt-4",
    "system_prompt": "You are a helpful assistant."
  }'

Start a Chat Session‚Äã

curl -X POST http://localhost:8080/api/v1/chat \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "message": "Hello, how can you help me?",
    "agent_id": "agent-123",
    "stream": true
  }'

WebSocket Streaming‚Äã

const ws = new WebSocket('ws://localhost:8080/api/v1/chat/stream');
ws.onopen = function() {
  ws.send(JSON.stringify({
    message: "Hello",
    agent_id: "agent-123",
    token: "YOUR_TOKEN"
  }));
};

Rate Limiting‚Äã

API requests are rate limited to:

  • 1000 requests per hour for authenticated users
  • 100 requests per hour for unauthenticated requests

Rate limit headers are included in all responses:

X-RateLimit-Limit: 1000
X-RateLimit-Remaining: 999
X-RateLimit-Reset: 1642248600
