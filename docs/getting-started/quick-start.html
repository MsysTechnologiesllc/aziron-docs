Quick Start

Get up and running with Aziron in just a few minutes! This guide will walk you through creating your first AI agent and sending your first request.

Prerequisites​

Make sure you have Aziron installed and running. If not, check out our Installation Guide.

Step 1: Verify Installation​

First, let's make sure Aziron is running:

curl http://localhost:8080/health

You should see a response like:

{
  "status": "healthy",
  "database": "connected",
  "redis": "connected",
  "uptime": "5m30s"
}

Step 2: Create a User Account​

Register a new user account:

curl -X POST http://localhost:8080/api/v1/auth/register \
  -H "Content-Type: application/json" \
  -d '{
    "username": "developer",
    "email": "dev@example.com",
    "password": "securepassword123"
  }'

Response:

{
  "status": "success",
  "message": "User registered successfully",
  "user_id": "uuid-here"
}

Step 3: Login and Get Token​

Login to get your authentication token:

curl -X POST http://localhost:8080/api/v1/auth/login \
  -H "Content-Type: application/json" \
  -d '{
    "username": "developer",
    "password": "securepassword123"
  }'

Response:

{
  "status": "success",
  "token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
  "expires_in": 86400,
  "user": {
    "id": "uuid-here",
    "username": "developer",
    "email": "dev@example.com"
  }
}

Save the token - you'll need it for authenticated requests:

export AZIRON_TOKEN="your-jwt-token-here"

Step 4: Create Your First Agent​

Now let's create a simple AI agent:

curl -X POST http://localhost:8080/api/v1/agents \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer $AZIRON_TOKEN" \
  -d '{
    "name": "HelloBot",
    "description": "A friendly greeting agent",
    "prompt": "You are a helpful AI assistant that greets users warmly and provides helpful information.",
    "model": "gpt-3.5-turbo",
    "max_tokens": 150,
    "temperature": 0.7,
    "capabilities": ["text_generation", "conversation"]
  }'

Response:

{
  "status": "success",
  "agent": {
    "id": "agent-uuid-here",
    "name": "HelloBot",
    "description": "A friendly greeting agent",
    "status": "active",
    "created_at": "2024-01-15T10:30:00Z"
  }
}

Step 5: Chat with Your Agent​

Send a message to your newly created agent:

curl -X POST http://localhost:8080/api/v1/agents/agent-uuid-here/chat \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer $AZIRON_TOKEN" \
  -d '{
    "message": "Hello! Can you introduce yourself?",
    "stream": false
  }'

Response:

{
  "status": "success",
  "response": {
    "message": "Hello! I'm HelloBot, your friendly AI assistant. I'm here to greet you warmly and help with any information you might need. How can I assist you today?",
    "agent_id": "agent-uuid-here",
    "timestamp": "2024-01-15T10:35:00Z",
    "tokens_used": 45
  }
}

Step 6: Try Streaming (Optional)​

For real-time responses, try streaming mode:

curl -X POST http://localhost:8080/api/v1/agents/agent-uuid-here/chat \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer $AZIRON_TOKEN" \
  -d '{
    "message": "Tell me a short story about AI",
    "stream": true
  }'

This will return a streaming response with real-time token generation.

Step 7: WebSocket Connection (Advanced)​

For full real-time interaction, connect via WebSocket:

// JavaScript example
const ws = new WebSocket('ws://localhost:8080/ws/agents/agent-uuid-here');
ws.addEventListener('open', function (event) {
    ws.send(JSON.stringify({
        type: 'auth',
        token: 'your-jwt-token-here'
    }));
});

ws.addEventListener('message', function (event) {
    const data = JSON.parse(event.data);
    console.log('Agent response:', data);
});

// Send a message
ws.send(JSON.stringify({
    type: 'message',
    content: 'Hello via WebSocket!'
}));

Step 8: List Your Agents​

View all your agents:

curl -X GET http://localhost:8080/api/v1/agents \
  -H "Authorization: Bearer $AZIRON_TOKEN"

Response:

{
  "status": "success",
  "agents": [
    {
      "id": "agent-uuid-here",
      "name": "HelloBot",
      "description": "A friendly greeting agent",
      "status": "active",
      "created_at": "2024-01-15T10:30:00Z",
      "last_used": "2024-01-15T10:35:00Z"
    }
  ],
  "total": 1
}
