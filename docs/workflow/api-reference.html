API Reference

Complete REST API reference for Aziron's workflow engine, including authentication, endpoints, request/response formats, and examples.

Base URL​

Production: https://api.aziron.com/v1
Staging: https://staging-api.aziron.com/v1
Development: http://localhost:8080/api/v1

Authentication​

JWT Token Authentication​

All API requests require authentication using JWT tokens:

# Login to get token
curl -X POST https://api.aziron.com/v1/auth/login \
  -H "Content-Type: application/json" \
  -d '{
    "email": "user@company.com",
    "password": "your_password"
  }'

# Response
{
  "token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
  "expires_in": 86400,
  "user": {
    "id": "user123",
    "email": "user@company.com",
    "roles": ["workflow_user"]
  }
}

# Use token in subsequent requests
curl -X GET https://api.aziron.com/v1/workflows \
  -H "Authorization: Bearer eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9..."

API Key Authentication​

For server-to-server communication:

curl -X GET https://api.aziron.com/v1/workflows \
  -H "X-API-Key: your_api_key_here"

Workflow Management​

List Workflows​

GET /workflows

Parameters:

  • page (integer): Page number (default: 1)
  • limit (integer): Items per page (default: 20, max: 100)
  • filter (string): Filter by name or description
  • sort (string): Sort field (name, created_at, updated_at)
  • order (string): Sort order (asc, desc)

Response:

{
  "workflows": [
    {
      "id": "email_processor",
      "name": "Email Processing Workflow",
      "description": "Process incoming emails with AI analysis",
      "version": "1.2.0",

      "updated_at": "2024-01-15T14:30:00Z",
      "created_by": "admin@company.com",
      "tags": ["email", "ai", "automation"],
      "execution_count": 1247,
      "success_rate": 0.97

  ],
  "pagination": {
    "page": 1,

  }

List Workflow Executions​

GET /workflows/executions

Parameters:

  • workflow_id (string): Filter by workflow ID
  • status (string): Filter by status (pending, running, completed, failed)
  • started_by (string): Filter by user
  • start_date (string): Filter executions after date (ISO 8601)
  • end_date (string): Filter executions before date (ISO 8601)
  • page (integer): Page number
  • limit (integer): Items per page

Response:

{
  "executions": [
    {
      "id": "exec_789xyz",
      "workflow_id": "email_processor",
      "status": "completed",
      "started_at": "2024-01-15T17:20:00Z",
      "completed_at": "2024-01-15T17:23:45Z",
      "duration": 225000,
      "started_by": "user@company.com",
      "success": true
    }
  ],
  "pagination": {
    "page": 1,
    "limit": 20,
    "total": 156,
    "total_pages": 8
  }
}

SDK Examples​

JavaScript/Node.js​

const AzironClient = require('@aziron/client');

const client = new AzironClient({
  baseURL: 'https://api.aziron.com/v1',
  token: 'your_jwt_token'
});

// Start workflow execution
const execution = await client.workflows.execute('email_processor', {
  email_content: 'Customer complaint...',
  customer_id: 'CUST-12345'
});

console.log('Execution started:', execution.id);

// Monitor execution progress
const status = await client.executions.get(execution.id);
console.log('Current status:', status.status);

Python​

from aziron_client import AzironClient

client = AzironClient(
    base_url='https://api.aziron.com/v1',
    token='your_jwt_token'
)

# Start workflow execution
execution = client.workflows.execute('email_processor', {
    'email_content': 'Customer complaint...',
    'customer_id': 'CUST-12345'
})

print(f'Execution started: {execution["id"]}')

# Monitor execution progress
status = client.executions.get(execution['id'])
print(f'Current status: {status["status"]}')

Go​

package main

import (
    "github.com/aziron/go-client"
)

func main() {
    client := aziron.NewClient(&aziron.Config{
        BaseURL: "https://api.aziron.com/v1",
        Token:   "your_jwt_token",
    })

    // Start workflow execution
    execution, err := client.Workflows.Execute("email_processor", map[string]interface{}{
        "email_content": "Customer complaint...",
        "customer_id":   "CUST-12345",
    })
    if err != nil {
        panic(err)
    }

    fmt.Printf("Execution started: %s\n", execution.ID)
}
